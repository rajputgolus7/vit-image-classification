# Vision Transformer Image Classification

## 🚀 Project Overview
This project uses Vision Transformer (ViT) for image classification on CIFAR-10 dataset.

## 🧠 Model
- Pretrained ViT (google/vit-base-patch16-224)
- Fine-tuned on CIFAR-10

## ⚙️ Tech Stack
- Python
- PyTorch
- Hugging Face Transformers

## 📊 Features
- Image classification using ViT
- Training and testing pipeline
- GPU accelerated (Google Colab)

## ▶️ How to Run

### Install dependencies
pip install -r requirements.txt

### Train model
python train.py

### Test model
python test.py
